# general-forms-poc

To install dependencies for backend:

```bash
npm install
```

To run backend:

```bash
npm run dev
```

To install dependencies for frontend:

```bash
cd frontend
npm install
```

To run frontend:

```bash
cd frontend
npm run dev
```

To store the data for Global Sections:

1. create a mysql database in local 
2. push the database using the command with prisma
